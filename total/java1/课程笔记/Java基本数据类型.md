# 基本数据类型（除了基本数据类型，其它都是引用数据类型）

## 1.布尔类型

 boolean，通常用来表示逻辑值，只有两个值：ture false

## 2.字符类型

 用的unicode编码格式，一个字符两个字节，16位二进制来表达，所以java中的char可以赋值汉字（详情见text2）

## 3.整数类型

1字节表示范围：-2^7到127

* byte，1字节
* short，2字节
* int，4字节
* long，8字节

## 4.浮点数据类型

* float 四字节 -3.403*10^38~3.403*10^38
* double 八字节 -1.798*10^308~1.798*10^308
