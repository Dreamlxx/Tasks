# 第一章题目

## 选择题部分

1. 一个完整的算法应具有“有穷性、可行性和确定性”
2. 计算其所处理的数据一般有某种关系，这是指“数据元素和数据元素”之间存在的某种的关系；我们把这种关系称之为“结构”
3. ![alt text](<PicturE/屏幕截图 2025-12-27 112907.png>)

  递归方法在函数调用过程中需要维护调用栈（每次调用保存返回地址、局部变量等），如果递归深度很大，会占用较多内存（栈空间），并且函数调用本身也有额外的时间开销（参数传递、栈帧分配等）。迭代方法通常用循环实现，没有频繁的函数调用开销，也不会有栈溢出的风险，因此通常时空效率更高。

4. 逻辑上可将数据结构分为“线性结构”和“非线性结构”
5. 顺序存储中表示的数据之间的逻辑关系是由“存储位置”决定的
6. 算法的时间复杂度与“问题规模”有关
7. 链接存储表示中数据元素之间的逻辑关系是由“指针”表示的 
8. 一个递归算法必须包括“中止条件”和“递归部分”
